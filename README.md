# Saving Logístico com Python

Projeto em Python para estimar economia de custos logísticos a partir da
antecipação de coletas, considerando distância, tempo de trânsito e regras
financeiras por transportadora.

## O que o projeto faz
- Calcula distância rodoviária estimada entre origem e destino
- Estima dias de trânsito
- Calcula dias antecipados de coleta
- Estima savings de diárias e armazenagem

## Tecnologias
- Python
- Pandas
- Geopy

## Entrada
Planilha Excel com dados de cargas (exemplo na pasta `data`).

## Saída
Planilha Excel com colunas adicionais de savings e métricas logísticas.
